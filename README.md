## XCalc로 할 수 있는 계산

3+4 , 3-4 , 3*4 , 3/4 : 사칙연산

2^10 : 2의 10제곱 

2^0.5 : 2의 제곱근(루트2)

Sin(30), cos(30), tan(30) :삼각함수

PI : 3.141592  예약 상수
